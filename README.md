# RaceFinisherV2

RaceFinisherV2 is a project for a **FPV race control system** based on [Race-Finisher](https://github.com/janix12/Race-Finisher).

## Overview
This is a part of a **race control system** with a custom ESP32 hardware setup.

## Features
- Oled screen
- ESP Now 

## Hardware
- lolin32_lite
